
A)Installation du projet Luzhanqi sur éclipse
Pour pouvoir lancer notre programme, tout d’abord il faut disposer de l’ide d'éclipse et d’une version compatible et à jour de JDK2.
Une fois ces pré-requis obtenu on lance éclipse. Une fois ouvert dans éclipse on crée un nouveau projet.
On peut voir qu’il faut d’abord accéder à :
1.⇒"File"
2.⇒"Import..."
Il faut ensuite dérouler le menu
1.⇒"General"
2.⇒"Projects from Folders or Archives"
3.⇒"Next"
Ensuite il choisir :
1.⇒Le chemin du dossier du projet
2.⇒Valider
3.⇒"Finish"

B)Utilisation du programme :
B)1. Mode pédagogique
	Attention Les pions ont un placement prédéfini et pour pouvoir jouer, il faut choisir un de ses pions,
en cliquant une fois dessus et choisir la case dans laquelle elle peut se déplacer.
	Attention pour se déplacer il faut bien s’assurer que le déplacement est possible sur la case sélectionner et cela ne doit pas :
1. Être une case trop éloigné
2. Être la case du campement qui est déjà occupé
3. Être une case hors map
4. Être pendant le tour de l’ordinateur
